# Week 17 — Assessment (team)

The team runs a timed inject set; coaches grade as the White Team would. Mark ✅ / 🔁. Record team result
+ per-inject scores. **Feeds:** the capstone "injects on time" dimension and professionalism thread
([`../02-readiness-rubric.md`](../02-readiness-rubric.md)).

This is **half of the Phase 3 exit gate** (Week 19's mini-scrimmage is the other half).

---

## Part A — Timed inject set (observed)
Release 4–6 injects (mixed technical + business, ≥1 multi-part, staggered deadlines), with a light fire
mid-set.

| # | Task | ✅/🔁 |
|---|---|---|
| A1 | Every inject has an owner and is logged with its deadline | |
| A2 | Technical injects done correctly, verified, **no scored service broken** | |
| A3 | Every submission is a PDF, in the right place, **before the deadline** | |
| A4 | Multi-part inject addresses **every** part | |
| A5 | Deadlines held even when the fire competed for attention | |

**Pass A** = ≥ 90% of injects submitted on time to standard, no self-inflicted outage.

## Part B — Quality (coach grades as White Team)
Grade each submission 1–5 on: format, completeness, specificity, professional tone, correct submission.

| # | Task | ✅/🔁 |
|---|---|---|
| B1 | Average quality ≥ 4/5 across the set | |
| B2 | Business responses read like a professional wrote them (tone, no slang, no team-ID) | |

**Pass B** = quality bar met.

---

## Scoring
- Pass A + B → the inject half of the Phase 3 gate is met.
- 🔁 → common misses: a finished inject submitted late (A3/A5 — fix the deadline clock), skipped parts
  of a multi-part (A4), or informal tone (B2). Re-run a shorter set focusing on the gap.

> Injects are worth as much as uptime. A team that owns, clocks, and writes them well outscores a team
> that only defends — every time. Grade this as seriously as you grade the boxes.
