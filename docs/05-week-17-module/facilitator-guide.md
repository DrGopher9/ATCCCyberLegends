# Week 17 — Facilitator Guide: Injects (Workflow, Deadlines, Professional Writing)

**Rubric targets:** the capstone "injects on time" dimension and the professionalism thread

**Council lens:** the **White Team Judge** owns this week. Injects are a **scored 35–50%** — worth as
much as uptime (see [`../07-competition-reference.md`](../07-competition-reference.md)). Teams that treat
injects as an interruption to "the real work" leave half the points on the table. This week makes
injects a first-class, owned, clocked workstream.

**You need:** the inject templates ([`inject-templates.md`](../../CCDC-main-Matt_2026/CCDC-main/competition-tools/inject-templates.md)
and the packet templates in [`CCDC_2026/Injects/`](../../CCDC_2026/Injects/)), a way to submit **PDFs**

## How injects really work (teach this first)
- Injects arrive via the **NISE/Team Portal**; **responses are PDFs** submitted there; **times are CST**.
- Each has a **deadline** and a **point value** weighted by importance/complexity; some have multiple
  parts, each scored.
- Two flavors: **technical** (add a user, stand up/migrate a service, produce a security config,
  password policy) and **business** (a memo to the "CEO," an incident report, a risk assessment,
  meeting minutes).
- Scored on completion **and quality** — clarity, thoroughness, professionalism. Missing the deadline =
  zero. A sloppy but complete inject still loses points.
- Running inject scores are **not shared** during the event — you never know your standing, so you can't
  coast.

## Learning objectives
By end of the week, the team can:
1. Run the inject workflow: receive → assign an owner → execute → **write to White-Team standard** →
   submit as PDF before the deadline → log it.
2. Write professionally: a memo/report a judge would score well.
3. Balance injects against defense — the **captain and injects owner protect inject time** so deadlines
   aren't missed while everyone's heads-down on boxes.

## Weeknight session plan (2–3 hr)
- **0:00–0:15 — Warm-up:** the incident-call format + a quick First-30 status drill (keeps Week 16 warm).
- **0:15–0:40 — Frame injects:** the workflow above; the scoring weight; show a **good vs. a weak**
  inject response (specific/professional vs. vague/informal). Walk the templates.
- **0:40–1:00 — The clock:** how injects compete with fires; the injects owner tracks deadlines and
  pulls people to injects *before* they're late; the captain arbitrates priority.
- **1:00–2:15 — Guided lab:** run [`lab-exercises.md`](lab-exercises.md) — a timed set of mixed
  technical + business injects, submitted as PDFs, then peer/coach-graded on the White-Team rubric.
- **2:15–2:30 — Debrief + assign [`homework.md`](homework.md); preview Week 18 (docs under load).**

## Weekend lab plan (3–5 hr)
- **Warm-up (20m):** review a graded inject from the weeknight — what would a judge dock?
- **Timed inject set (90m):** release 4–6 injects with staggered deadlines (some technical requiring
  real box work, some business writing, at least one multi-part). The team assigns owners, executes,
  and submits PDFs on time. Injects owner tracks the board; captain balances against a couple of light
  fires the facilitator introduces.
- **White-Team grading (30m):** coaches grade each submission on clarity/thoroughness/professionalism
  and on-time; return scores and specific feedback.
- **Assessment (30m):** run [`assessment.md`](assessment.md).
- **AAR (15m).**

## Facilitator notes & common snags
- **Injects feel like a distraction to new teams.** Reframe relentlessly: injects are *half the score*.
  Praise a well-written memo like you'd praise evicting the red team.
- **Deadlines are the killer.** The most common loss is a finished inject submitted late, or forgotten
  under fire. Train the injects owner to run a visible deadline clock.
- **Quality is teachable.** Give a rubric: correct format, addresses every part, specific, professional
  tone, correct submission (PDF, right place). Grade to it consistently.
- **Technical injects still need verification.** "Add a user and configure X" — confirm it actually
  works and **don't break a scored service** doing it.
- **Business-persona tone.** Responses go to a simulated CEO/manager — professional, no team-identifying
  info beyond what's asked, no slang.

## Definition of done
The team completed a timed, mixed inject set — owned, executed, written to standard, submitted as PDFs
on time — with the injects owner running the clock and the captain balancing against fires. That's the
first half of the Phase 3 exit gate (the mini-scrimmage in Week 19 is the second).
