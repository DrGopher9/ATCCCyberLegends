## Scoring
- Pass A + B → the team enters Week 22 (heavier) with Week-20's gaps actually fixed.
- 🔁 → a fix that's only "discussed" is not closed; carry it forward explicitly and make it a priority in

> next scrimmage surfaces *new*, higher-level problems — not the same ones again.
