# Week 12 — Assessment

Member **performs** on **their own box** while observed. Mark ✅ / 🔁. Record in the rubric sheet.
**Covers:** the baseline foundation for the member's Tier-3 track (no gate yet — the Tier-3 gate is
Week 15).

---

## Part A — Baseline mastery (hands-on + verbal, unaided)
Time budget: **8 minutes.**

| # | Task | ✅/🔁 |
|---|---|---|
| A1 | State your box's normal listeners, processes, and users from your baseline | |
| A2 | Name your scored component(s) and demonstrate the command that proves each works | |
| A3 | Break one scored service and recover it; verify it serves | |
| A4 | Show your config backup and explain how you'd restore from it | |
| A5 | Name two attacks you expect on this box and where you'd see each | |

**Pass A** = all five ✅; the baseline is specific (real ports/accounts, not vague).

## Part B — Backup depth
| # | Task | ✅/🔁 |
|---|---|---|
| B1 | Your cross-training backup can describe your box's scored service and how to recover it | |

**Pass B** = the backup can speak to the box (two people know it).

---

## Scoring
- Pass A + B → ready for the red team in Week 13.
- 🔁 on A → usually a vague baseline or a slow/failed recovery. Give reps; a shaky baseline means missed
  attacks next week.
- 🔁 on B → seat the backup with the primary again before Week 13.

> No Tier-3 items are *awarded* this week — Week 12 builds the "normal" you'll defend deviations from.
> The payoff lands in Weeks 13–15 when the attacks start.
