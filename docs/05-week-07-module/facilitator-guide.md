# Week 7 — Facilitator Guide: Foundations Checkpoint (Tier-1 Gate + Review)

> **Phase 0, Week 7 — the exit gate.** Built from [`../templates/module-template.md`](../templates/module-template.md).

**Rubric target:** the full **Tier 1** in [`../02-readiness-rubric.md`](../02-readiness-rubric.md) —
`T1-G1`, `T1-G2`, `T1-L1`–`L4`, `T1-W1`–`W2`, `T1-N1`. Passing Tier 1 is what unlocks Phase 1.

**Council lens:** the Coach owns this week. It's a checkpoint, not a filter — the goal is to get
*everyone* to Tier 1 (or onto a clear remediation plan), not to rank people. Early, visible mastery is
the retention lever heading into the harder phases.

**You need:** lab access for all boxes used across Weeks 1–6, the per-member rubric-tracking sheet with
everyone's running ✅/🔁 marks, and the [`assessment.md`](assessment.md) as the gate.

---

## Learning objectives
This week has no new content. By end of the week, every member either:
1. **Passes the full Tier-1 gate**, or
2. Leaves with a **named partner and a written remediation plan** for the specific items they missed.

## Weeknight session plan (2–3 hr) — Review circuit
- **0:00–0:10 — Frame it:** this is a checkpoint, not a cut. Everyone's target is Tier 1. Collect the
  Week 6 self-checks and note each member's circled 🔁 items.
- **0:10–2:15 — Stations (rotate):** run [`lab-exercises.md`](lab-exercises.md) as a review circuit —
  five stations covering Linux ops, Linux service recovery, Windows admin+events, networking+topology,
  and git+rules. Members spend the most time on their own 🔁 items; strong members coach at stations
  they've mastered (teaching cements it).
- **2:15–2:30 — Set expectations for the weekend gate; assign any last drilling.**

## Weekend lab plan (3–5 hr) — The gate
- **Warm-up (20m):** quick pass of each member's weakest item.
- **Tier-1 gate (2–2.5 hr):** run [`assessment.md`](assessment.md) — the full Tier-1 checklist,
  performed, not described. Two assessors if possible so it moves. Record ✅/🔁 per item per member.
- **Remediation planning (30m):** for anyone not fully ✅, write a specific plan (which items, which
  partner, by when — ideally closed in the first week of Phase 1). See [`homework.md`](homework.md).
- **Phase 0 AAR + celebration (20m):** acknowledge how far a brand-new team has come in 7 weeks.
  Preview Phase 1 (Defensive Core).

## Facilitator notes & common snags
- **`T1-L3` (service recovery) is the usual sticking point.** Budget extra reps at that station.
- **Don't fail people quietly.** A 🔁 is a plan, not a verdict. The bench trains as equals; Tier-1
  status is about readiness, and there's runway to close gaps.
- **Peer coaching is your force multiplier.** Members who've mastered a station teaching it is both
  good pedagogy and good scrimmage-team behavior.
- Record everything in the tracking sheet — Tier-1 status per member is the data you'll use to shape
  Phase 1 pairings.

## Definition of done
Every member is either **Tier-1 ✅** or has a **written, owned remediation plan** with a close date in
early Phase 1. Phase 0 exit gate met when the team as a whole is at (or credibly closing on) Tier 1.
