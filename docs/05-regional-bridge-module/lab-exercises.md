# Regional Bridge — Drills

A compressed re-peak for the Midwest Regional. Structure mirrors Phase 4 (scrimmage → AAR → close gaps)
at a higher ceiling. Drive it from the **qualifier AAR**. Use the
[scorecard](../templates/scrimmage-scorecard.md) and [AAR template](../templates/aar-template.md).

---

## Block A — Close qualifier gaps + raise intensity
- [ ] Put the qualifier AAR's top issues on the board with owners
- [ ] Close them (demonstrated, per the [Week-21 fix menu](../05-week-21-module/lab-exercises.md))
- [ ] Full scrimmage with the red team **above qualifier intensity** (more footholds, faster injects,
      aggressive lateral movement); score it; AAR

## Block B — Deepen + scale
- [ ] **Bench depth:** every box now three-deep if possible (regionals run longer/harder) — backups run
      boxes solo again under harder pressure
- [ ] **New services/boxes:** if the regional environment adds any, baseline + assign them fast
      (compressed Phase-2 pattern): normal state, scored component, break/recover, back up
- [ ] **Faster injects:** drill a heavier, tighter inject load
- [ ] Full scrimmage; AAR

## Block C — Dress rehearsal + re-taper
- [ ] Full-length, **regional-intensity** dress rehearsal against the strongest red team; score to the
      capstone bar
- [ ] Re-lock logistics for the regional venue (onsite/travel, connectivity, kit, roster, confirm
      repo/web-proxy + rules for the regional)
- [ ] **Re-taper:** light maintenance + rest into the event; factor travel fatigue
- [ ] Final brief (as Week 27), adjusted for the regional environment

## Done?
Ready for the regional when qualifier gaps are closed, the team holds the capstone bar under
above-qualifier intensity, any new services are covered, the bench is deeper, and the team re-tapered
into the event. Individual prep: [`homework.md`](homework.md).
