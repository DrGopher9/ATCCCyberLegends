# Week 19 — Facilitator Guide: First Mini-Scrimmage + AAR (Phase 3 Exit)

> **Phase 3, Week 19 — the phase exit gate.** Built from [`../templates/module-template.md`](../templates/module-template.md).

**Rubric targets:** the Phase 3 exit gate — the team completes a **timed inject set** (Week 17) **and
survives a mini-scrimmage**, then runs an honest AAR. First real look at the capstone dimensions in
[`../02-readiness-rubric.md`](../02-readiness-rubric.md).

**Council lens:** all five, but with guardrails — the **Coach** keeps it *gentle* (this is the team's
first contact; the goal is to normalize getting attacked, not to crush morale), the **Red Teamer**
brings light, bounded pressure, the **White Team** scores injects + docs, and the **Former Tech Lead**
watches whether the team actually operates as one.

**You need:** the full lab, the live red team (dialed to **gentle**), a short inject set, the
[scrimmage scorecard](../templates/scrimmage-scorecard.md) and [AAR template](../templates/aar-template.md),
and 3–4 hours for a compressed scrimmage + a real debrief.

---

## What a mini-scrimmage is
A **compressed, low-intensity** dress rehearsal — think **2–3 hours**, not the full 7 — that puts
everything from Phase 3 together at once: First-30, roles/comms, injects on a clock, docs under load,
detection + IR, all while a gentle red team pokes at the boxes. It is scored, but the score is a
**baseline to improve from**, not a pass/fail cut. Phase 4 ramps the intensity from here.

## Learning objectives
By end of the week, the team can:
1. Run a compressed scrimmage: open with the First-30, hold scored services, complete injects on time,
   detect + evict light red-team activity, keep clean docs — as one coordinated unit.
2. Produce an honest **AAR** and a top-3 fix list to carry into Phase 4.

## Weeknight session plan (2–3 hr) — rehearsal & readiness
- **0:00–0:20 — Warm-up:** a full team First-30 status pass (keeps Week 16 sharp).
- **0:20–0:45 — Pre-brief the scrimmage:** confirm roles, captain, injects owner, docs owner, the comms
  protocol, and the priority order. Set expectations: gentle, a baseline, learn from the AAR.
- **0:45–2:15 — Mini-drill:** a 60–75 min compressed run (lighter than the weekend) so the mechanics of
  a scored scrimmage are familiar before the real one. Short AAR.
- **2:15–2:30 — Assign [`homework.md`](homework.md) (pre-scrimmage prep); preview the weekend scrimmage.**

## Weekend lab plan (3–5 hr) — the mini-scrimmage + AAR
- **Set-up (20m):** boards up, scorecard ready, roles confirmed, red team briefed to *gentle*.
- **Mini-scrimmage (2–2.5 hr):** run [`lab-exercises.md`](lab-exercises.md) — First-30, then hold
  services + injects + light incidents. Facilitators score on the [scorecard](../templates/scrimmage-scorecard.md).
- **AAR (40m):** run the full [AAR template](../templates/aar-template.md) — what was scored, what the
  red team achieved, what went well, root causes, **top-3 fixes with owners/dates**. Blameless.
- **Phase 3 close (15m):** celebrate the team's first live-fire; preview Phase 4 (higher intensity, real
  AAR loop).

## Facilitator notes & common snags
- **Keep it genuinely gentle.** The failure mode here is a red team that goes too hard and demoralizes a
  team that's only been integrated for four weeks. Dial pressure to "they'll catch most of it if they're
  paying attention."
- **Score to learn, not to judge.** Share the scorecard openly in the AAR — it's the baseline Phase 4
  improves on. Don't rank individuals.
- **Watch the seams, not the boxes.** The interesting failures are team failures: an inject missed
  because everyone was heads-down on a fire, an incident nobody owned, a stale board. Those are the
  top-3 fixes.
- **The AAR is the point.** A scrimmage without an honest AAR just rehearses mistakes. Protect the AAR
  time; make it blameless and specific.

## Definition of done (Phase 3 exit gate)
The team completed a timed inject set (Week 17) **and** a mini-scrimmage as a coordinated unit, and
produced an honest AAR with an owned top-3 fix list. Rough edges are expected and welcome — they're the
Phase 4 agenda. The team is ready to move from *integrating* to *getting good under fire*.
