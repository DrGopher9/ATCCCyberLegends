# Week 26 — Taper Activities & Logistics Runbook

Not a lab this week — a logistics checklist plus **light** maintenance. The goal is to arrive at the
qualifier sharp and rested, with everything staged. Confirm details against the current packet /
[`../07-competition-reference.md`](../07-competition-reference.md).

---

## 1. Lock the roster
- [ ] Starting **8** chosen on Phase 1–4 checklist + scrimmage performance (min 4, **max 2 grad
      students**)
- [ ] Alternates confirmed on the 12-person roster; each knows their sub-in role
- [ ] Every box has a designated primary **and** backup (Week-23 grid)
- [ ] Captain, injects owner, docs owner confirmed

## 2. Lock logistics
- [ ] Competition date + start time confirmed — **times are CST** (2026 was Jan 31, 9am–4pm; confirm
      2027)
- [ ] Onsite or remote? If remote, connectivity meets requirements; if onsite, travel/setup planned
- [ ] Competition accounts / portal access confirmed
- [ ] What's allowed in-room confirmed (printed materials **yes**; USB/personal devices generally **no**)

## 3. Submit the GitHub repo URL (don't skip)
- [ ] Send the team repo URL to the **State Director before the event** so it's reachable via the
      competition web proxy on game day (packet rule)
- [ ] Confirm receipt in writing

## 4. Assemble the in-competition kit (print/stage)
Work the kit checklist in [`facilitator-guide.md`](facilitator-guide.md). Print everything — you can't
rely on digital access, and printed reference is explicitly allowed.

## 5. Light maintenance drills (keep sharp, don't grind)
One crisp rep of each — no marathon:
- [ ] Team First-30 (timed, clean)
- [ ] Credential sweep on one box
- [ ] Service recovery on one box
- [ ] One IR report to spec
- [ ] A few comms calls in format

## 6. Start protecting energy
- [ ] Plan rest for the final week — no late-night cramming
- [ ] Confirm the day-of energy plan (breaks, food, backup rotations for a 7-hour day)

## Done?
Week 26 is complete when the roster/roles/logistics are locked, the GitHub URL is submitted, the kit is
assembled, and the team stayed sharp on light drills without exhausting itself. Individual prep:
[`homework.md`](homework.md).
