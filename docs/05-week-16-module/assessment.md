# Week 16 — Assessment (team + individual)

A team assessment with individual role checks. Mark ✅ / 🔁. Record team result + per-member notes.
**Feeds capstone dimensions:** "first 30 minutes," "communication," "no self-inflicted outages"
([`../02-readiness-rubric.md`](../02-readiness-rubric.md)).

---

## Part A — Team: First 30 Minutes (observed, timed)
The team runs the opening runbook as a unit. Time budget: **30 minutes.**

| # | Task | ✅/🔁 |
|---|---|---|
| A1 | Every box: credential sweep + persistence sweep completed | |
| A2 | Every box: firewall up, scored service verified, ICMP preserved | |
| A3 | Every box: forwarder confirmed reporting | |
| A4 | Every action logged on the shared board | |
| A5 | **No scored service dropped** during the drill (no self-owns) | |

**Pass A** = all boxes done inside 30 min with no self-inflicted outage.

## Part B — Comms & captain (observed)

| # | Task | ✅/🔁 |

---

## Scoring
- Team passes A + B, most members pass C → team-ops foundation is in place for injects (Week 17) and
  the mini-scrimmage (Week 19).
- 🔁 → note whether it's a *coordination* gap (comms/captain) or an *individual* gap; drill accordingly.
  First team reps are rough — the point is to surface the gaps now, not to be perfect.

> The measure of this week isn't individual skill (you proved that in Phases 1–2) — it's whether eight
> specialists move as one. A calm captain and clean comms are worth more points in a 7-hour day than any
> single clever fix.
