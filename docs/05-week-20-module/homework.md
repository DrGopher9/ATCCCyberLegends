# Week 20 — Homework

## Before the scrimmage (~30 min)
- [ ] Re-confirm your box's First-30 checklist, config backup, detections, and service-verify command
- [ ] Read the capstone dimensions on the [scorecard](../templates/scrimmage-scorecard.md) so you know
      what's measured
- [ ] Note which Week-19 gap you're personally responsible for improving

## After the scrimmage (~30 min) — follow-through into Week 21
- [ ] Read the scorecard: our weakest dimension this run was ______
- [ ] Did the moderate red team get into your box? If so: did you detect it, how long until eviction,
      did eviction stick, did you file a report?
- [ ] Your biggest personal takeaway (one thing you'll do differently)
- [ ] If you own a top-3 fix, write the plan + due date (Week 21 closes it)

## Bring to Week 21 (fix-it)
- The AAR top-3 fix list
- Your box's specific weakness from this scrimmage
- Your owned fix + plan
